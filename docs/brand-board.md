# Empire Fence Brand Board

## Direction

- Tone: premium contractor, calm confidence, structured, not luxury-for-luxury
- Reference language: `Aethar` frame discipline + `Archline` editorial spacing and rhythm
- Surface model: dark framed shells over warm stone content areas
- Visual priority: image-led, high contrast, restrained bronze accents

## Core palette

- `#081018` — midnight base
- `#111b26` — deep navy surface
- `#24374b` — framed panel blue
- `#f3ede4` — warm canvas
- `#fbf7f0` — bright content surface
- `#c89a6b` — bronze accent
- `#e3d6c4` — muted stone highlight
- `#12161d` — primary text

## Usage rules

- Dark shells for hero, framed page intros, booking sections, and sticky header
- Warm canvas for the main content field
- Bronze only for:
  - eyebrows
  - primary CTAs
  - focus states
  - small accent rules
- Avoid blue accents outside structural shells

## Typography

- Display: `Cormorant Garamond`
- Body: `Inter`

### Hierarchy

- Hero display: very large, tightly tracked serif
- Section titles: serif, compressed spacing, short lines
- Body: restrained, high readability, no marketing fluff
- Labels / eyebrows: uppercase, tracked, bronze

## Spacing

- Section padding: `78–132px`
- Card padding: `22–32px`
- Grid gaps:
  - primary layouts: `24–56px`
  - secondary cards: `18px`

## Motion

- Curve: `cubic-bezier(0.16, 1, 0.3, 1)`
- Reveal: opacity + upward translation only
- Hover: subtle `translateY(-2px)`
- No decorative parallax unless it clearly improves the hero

## Design constraints

- Avoid generic SaaS gradients
- Avoid glassmorphism as the main language
- Avoid overly bright gold or beige luxury clichés
- Keep imagery dominant where possible
- Keep service components modular enough for page reuse
